ViewBovis is a web application for exploring whole genome sequencing (WGS) data from Mycobacterium bovis. By linking genetic relatedness with geographical neighbourhood, you will be able to understand the transmission of this disease in both bovine and non-bovine animals. Please note the scheduled downtime for ViewBovis is from 10pm-8am daily.

SNP Distance: SNP stands for Single Nucleotide Polymorphism. SNPs are single changes in a nucleotide at a specific point in the genome (e.g. A -> T). SNP Distance is the number of differences between two samples; the fewer SNP differences, the more closely related the samples are.

SNP Map

This map will allow you to pick a sample of interest and view related samples using a SNP distance threshold.


SNP Matrix

Visualise a pairwise SNP matrix containing a target sample and related samples. Each pairwise comparison displays the SNP difference between two samples.

Nextstrain is a tool used to visualise whole genome sequencing (WGS) data from Mycobacterium bovis. You can view and explore both a phylogenetic tree and a map to understand genetic relatedness and locate samples geographically. This visualisation enables you to view data on both bovine and non-bovine samples.

Search & Select Data

Use the search box to find your sample of interest. Click a link in the table to view the phylogenetic tree in Nextstrain.

  • Clicking an Identifier (e.g. Ear Tag) or Submission (e.g. AF number) highlights the sample of interest for that clade.
  • Clicking on CPH or County highlights all samples in the same CPH or County, respectively.
  • Clicking on the Clade or the link to the 'Commonly Viewed Clades' shows the whole phylogenetic tree.

For example: Identifier (Ear Tag: UK262728200982) OR Submission (AF number: AF-21-08014-22).
